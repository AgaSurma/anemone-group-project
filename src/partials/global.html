Na tej stronie macie gotowe komponenty w wersji mobilnej do wklejenia do waszych sekcji/zadań. Są to komponenty globalne, wspólne

Wklejajcie takie elementy jakie tu są (tagi) oraz klasy nadawajcie takie jak tutaj są nadane.

Komponenty globalne: (w wersji mobilnej)

Element button z klasą css .button

Element svg z klasą .logo

Element label z klasą .form-field posiadający wewnątrz element input z klasą .form-field__text-input (label.form-field > input.form-field__text-input)

Powinien reagować zmianą styli na zgodność wprowadzonych danych
